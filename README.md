# Eternal Loyalty

## Description

Platform game delevoped for a subject in the career of Programming and Desing in videogmaes - UPC. 

The game is about a dog trying to arrive to his owner.

## Key Features

 - Jumps.
 - Fall into the void.
 - Kind of flying.
 - Go forwards and backwards.
 - Animations.
 - Colliders.
 - Die.
 - Textures.
 
## Controls

 - To move it is used W-A-S-D.
